## 🚀 Cara Menjalankan Proyek  

1. **Clone Repository**  
   ```bash
   git clone https://github.com/DeJavi08/Portfolio.git
   ```
2. **Masuk ke Direktori Proyek**  
   ```bash
   cd Portfolio
   ```
3. **Jalankan di Localhost** (Jika menggunakan React)  
   ```bash
   npm install
   npm start
   ```
